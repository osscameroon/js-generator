# How to Contribute

```shell
# 1. Clone
git clone git@github.com:osscameroon/js-generator.git

# 2. Move to root directory
cd js-generator

# 3. Tests & Build
mvn clean package

# 4. Browse through code
# 5. Run CLI with --help and play with it
# 6. Fork the project, build, test, open a pull request
```
